# Gyro Homebrew Tap

This tap distributes the open-source [`gyro` CLI](https://github.com/wytzeh197/Gyro). It does not install Gyro.app.

```bash
brew tap wytzeh197/tap
brew install gyro
```

Verify the installation with `gyro --version` and `gyro doctor --json`.

Download Gyro.app from the [Gyro macOS site](https://wytzeh197.github.io/Gyro/).
